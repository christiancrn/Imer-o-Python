# Projeto: JS Curso - Aula 4

Pequeno projeto Web desenvolvido durante as aulas de lógica e programação em JavaScript. Contém uma página principal e um jogo exemplo na pasta `img`.

## Recursos
- Página estática com HTML, CSS e JavaScript.
- Exemplo de jogo em `img/JS Game.html`.

## Estrutura do projeto

- [index.html](index.html) — página principal do projeto
- [app.js](app.js) — script principal
- [style.css](style.css) — estilos principais
- img/ — pasta com recursos e um jogo exemplo (`JS Game.html`)

## Como executar

Opções rápidas:

- Abrir `index.html` diretamente no navegador.
- Servir a pasta localmente (recomendado para funcionalidades que exigem servidor):

```bash
cd "d:\\Projetos de aulas tecnologia\\logica programação\\js-curso-2-aula_4"
python -m http.server 8000
# depois abra http://localhost:8000 no navegador
```

Se estiver usando VS Code, instale e use a extensão Live Server para abertura rápida.

## Observações

- Não há dependências externas; é um projeto front-end simples.
- O arquivo do jogo está em `img/JS Game.html` com seus próprios arquivos em `img/JS Game_files/`.
